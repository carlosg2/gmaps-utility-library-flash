Allows you to filter the features of individual layers in the exported map by specifying definition expressions for those layers. Syntax: layerId1:layerDef1;layerId2:layerDef2 Example: 0:POP2000 > 1000000;5:AREA > 100000 Normally you do not set this property directly. Instead, use Layer.definition for each layer.
