Classpublic class GradientRule
InheritanceGradientRule Range
SubclassesGradientRuleList

GradientRule is a basic building block for gradient configurations. One GradientRule object corresponds to a gradient transition between two color values. Together with other GradientRule objects, it sholud be contained in GradientRuleList to form a full gradient config.
