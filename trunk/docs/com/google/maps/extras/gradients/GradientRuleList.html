Packagecom.google.maps.extras.gradients
Classpublic class GradientRuleList
InheritanceGradientRuleList GradientRule Range

GradientRuleList is the top class for creating gradient configs. It can be constructed from a list of GradientRule objects or from an XML spec. When combined with a list of Numbers, GradientRuleList produces a GradientControl for subsequent use.
