Packagecom.google.maps.extras.arcgislink
Classpublic dynamic class Layer
Implementsflash.events.IEventDispatcher

This class represents a Map layer inside an ArcGISMapService. It carries information about a layer's name, id and other information such as scales etc. Due to the way REST API is implemented, each layers extra meta data must retrieved individually. However, most operations do not require those extra info and can be used directly.
