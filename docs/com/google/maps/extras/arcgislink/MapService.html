Packagecom.google.maps.extras.arcgislink
Classpublic dynamic class MapService
Implementsflash.events.IEventDispatcher

This class is the core class for all map service operations. It represents an Server map service and serve as the underline resource represented by ArcGISTileLayer and ArcGISMapOverlay. It is constructed asynchronously so it should be used after it is loaded, either by handle its "load" event, or used in a callback function passed in the constructor.
For more info see Map Service

Creates a ArcGISMapService objects that can be used by UI components.

  • url (required) is the URL of the map servive, e.g. http://server.arcgisonline.com//rest/services/ESRI_StreetMap_World_2D/MapServer.
  • opt_service optional parameter of type ArcGISMapServiceOptions
      Note the spatial reference of the map service must already exists in the ArcGISSpatialReferences if actual coordinates transformation is needed.
